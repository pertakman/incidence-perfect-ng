## What Changed (Highlights)

- Added board-operator battery presence mode in web `Network` panel:
  - `Auto detect`
  - `Installed`
  - `No battery installed`
- Battery mode now persists in NVS (`Preferences`) across reboot.
- If `No battery installed` is selected, battery indicators are hidden in:
  - device touch UI
  - web UI
- Web control layout harmonized with touch UI order.
- Guided web action button order kept consistent with common web patterns:
  - `CANCEL` left, `CONFIRM/CAPTURE` right

  - `T55` battery mode set to `No battery installed` and persisted
  - `T56` persistence confirmed across power cycle
  - `T57` battery indicators restored when returning to `Auto`/`Installed`

## Notes / Limitations

- Battery presence auto-detection remains heuristic on this hardware due to shared USB/battery rail behavior.
- `No battery installed` mode is recommended for mains/USB-only installations.
