## What Changed Since RC2

- Boot splash now follows the stored `ROTATE` orientation instead of appearing upside down after a rotated reboot.
- Added persistent `Startup ZERO` control in the web `Network` panel:
  - default is enabled on cold boot
  - persists across reboot
  - can be disabled when the operator wants to keep the saved zero reference unchanged at startup
- Network settings action wording is clearer:
  - `Save Network` is now simply `Save`

## Validation Focus For RC3

- Boot UX:
  - splash orientation matches stored `ROT 0` / `ROT 180`
  - firmware version overlay remains readable in both cases
- Startup ZERO:
  - `Enabled` starts ZERO automatically on cold boot
  - `Disabled` suppresses auto-ZERO on cold boot
  - setting persists across power cycles
- Web smoke:
  - `Network` panel still saves settings correctly
  - wording changes do not affect behavior
